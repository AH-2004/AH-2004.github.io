Emphasis

You can emphasize text using asterisks or underscores.

  • Italic text can be created with one asterisk or underscore on each side.
  • Bold text can be created with two asterisks or underscores on each side.
  • Bold and Italic text can be created with three asterisks or underscores on each side.
